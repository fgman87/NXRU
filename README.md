# NXRU
 Learning Godot


Movement:
    A: Input action "left"
    D: Input action "right"
    W: Input action "up"
    S: Input action "down"

Combat:
    1: Equip Bow
    2: Equip Sword
    3: Equip Lightning (nothing atm)
    4: Equip Fire (fireball) requires 100 resource
    Left Mouse Button: Attack
